\# KOSKİ Vatandaş Destek Chatbotu



Bu proje, KOSKİ için geliştirilmiş temel seviyede bir vatandaş destek chatbot uygulamasıdır.



Proje, tek seferde tamamlanmış bir yapı olarak değil, aşama aşama geliştirilmiş ve her aşama Git commitleri ile kayıt altına alınmıştır.



\## Projenin Amacı



Vatandaşların KOSKİ hizmetleri hakkında dijital asistan üzerinden destek almasını sağlamak ve personelin vatandaş görüşmelerini yönetim panelinden takip edebilmesini amaçlar.



\## Geliştirme Aşamaları



1\. Git ve GitHub ayarları yapıldı.

2\. Ana giriş portalı eklendi.

3\. Vatandaş chatbot arayüzü eklendi.

4\. Personel giriş ekranı eklendi.

5\. Personel yönetim paneli eklendi.

6\. FastAPI backend yapısı eklendi.

7\. PostgreSQL veritabanı işlemleri eklendi.

8\. Ollama yapay zeka bağlantısı eklendi.

9\. Ortam değişkenleri ve güvenlik düzenlemeleri yapıldı.

10\. Python bağımlılıkları `requirements.txt` dosyasına eklendi.



\## Kullanılan Teknolojiler



\- HTML

\- CSS

\- JavaScript

\- Python

\- FastAPI

\- PostgreSQL

\- Ollama

\- Git ve GitHub



\## Not



Gerçek şifreler ve kişisel bilgiler `.env` dosyasında tutulur. `.env` dosyası GitHub’a yüklenmez.
